# PythonMeetup Bot

Telegram-бот для сопровождения митапов, конференций и других мероприятий.

Бот позволяет участникам просматривать программу мероприятия, задавать вопросы спикерам, отправлять заявки на выступление, получать уведомления от организатора, а также поддерживать проект через систему донатов.

---

# 🚀 Возможности

## Для участников

- 📅 Просмотр расписания мероприятия
- 🎤 Просмотр информации о текущем спикере
- ❓ Отправка вопросов спикеру
- 📝 Подача заявки на выступление
- 📢 Получение уведомлений от организатора
- ❤️ Поддержка проекта (донат)

## Для организатора

- 👤 Назначение текущего спикера
- ➕ Добавление новых событий
- ➖ Удаление событий
- 📜 Просмотр расписания
- 📝 Просмотр заявок на выступление
- 📢 Массовая рассылка уведомлений участникам
- 🧹 Очистка расписания

# 📦 Установка

## 1. Клонировать репозиторий

```bash
git clone <URL_репозитория>
cd <папка_проекта>
```

## 2. Создать виртуальное окружение

### Windows

```bash
python -m venv .venv
.venv\Scripts\activate
```

### Linux / macOS

```bash
python3 -m venv .venv
source .venv/bin/activate
```

## 3. Установить зависимости

Если в проекте присутствует файл `requirements.txt`:

```bash
pip install -r requirements.txt
```

Или установить вручную:

```bash
pip install aiogram python-dotenv
```

---

# ⚙️ Настройка

Создайте в корне проекта файл `.env`.

```env
BOT_TOKEN=YOUR_BOT_TOKEN
ORGANIZER_ID=YOUR_TELEGRAM_ID
PAYMENT_TOKEN=YOUR_PAYMENT_PROVIDER_TOKEN
```

## Описание переменных

| Переменная | Описание |
|------------|----------|
| `BOT_TOKEN` | Токен Telegram-бота, полученный через BotFather. |
| `ORGANIZER_ID` | Telegram ID пользователя, имеющего доступ к панели организатора (`/admin`). |
| `PAYMENT_TOKEN` | Токен платежного провайдера Telegram Payments. Используется для работы функции донатов. |

# 💬 Использование

### Для участников

После запуска отправьте команду:

```text
/start
```

### Для организатора

Для открытия панели администратора используйте:

```text
/admin
```

Доступ к панели организатора имеет только пользователь, чей Telegram ID указан в переменной `ORGANIZER_ID`.

---

# 📝 Примечания

- Все данные приложения хранятся локально в файле `db.json`.
- Перед запуском необходимо создать Telegram-бота через **BotFather** и указать его токен в файле `.env`.
- Для работы функции донатов необходимо подключить платежного провайдера через **BotFather** и получить `PAYMENT_TOKEN`.
- Если `PAYMENT_TOKEN` отсутствует или не настроен, функция донатов работать не будет.
- При первом запуске бот автоматически создаст необходимые структуры данных.
